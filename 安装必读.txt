【Project DIVA PC 4.0, HD v2.0】多国语音翻唱版

游戏遇到不支持的视频时，会自动 安装视频解码器 或 转换视频，永久性解决视频解码问题。
全面支持多国语言，大家可以看到美丽的阿拉伯语、韩语、日语等。但是除了中英文之外，其它国家的语言均为谷歌翻译，质量低下。


注意事项：
有玩家反映一些老map存在音频延迟问题，这可能是因为视频和音频没有分开。一般来说，视频延迟比音频大很多，视频一旦出现延迟，视频中的音频也会跟着延迟，而且不同电脑上视频延迟相差很大，所以Project DIVA的MAP在设计上要求视频和音频分开。
如果游戏选项中启用全局键盘钩，360safe等杀毒软件会报告木马程序监听键盘输入，盗取用户信息。
不用担心，没有木马。不放心的玩家可以自行用VS2010编译。




游戏包自带2首雪初音歌曲包《歌に形はないけれど》和《胧月》，以及《罗密欧与灰姑娘》（多服装版）
游戏改进如下：
1. 修正按键判定bug，提前0.25秒以上按键不会让后面的note消失
2. 设置中添加按键判定模式，分为标准（按错的键不会取消下个音键）和严格（判定时间内按错的键会取消下个音键），以适合不同玩家习惯
3. 更新了整个按键处理系统，大幅度减少按键延时，同时支持将多键映射为同一键（适合玩观赏型map）
4. 更新了图形系统画长条键的算法，大幅度提高长条键的绘画速度，（需在选项中将反锯齿设为1）
5. 设置中添加自定义按键设定功能
6. 设置中添加之前几套皮肤，现共6套，支持所有原始经典皮肤（适合怀旧的玩家）
7. 设置中皮肤选项中开启大写键可以预览皮肤
8. 粒子系统引入“硬件自适应调控技术”，画面效果自动适应硬件性能。也就是说：电脑配置越高，画面效果越好；电脑配置再差，画面也不至于太卡。
9. 游戏一开始按<Esc>可以跳过片头
10. 歌曲菜单界面按F5直接刷新歌曲列表，无需重启游戏
11. 增加界面字体选项，支持宋体、楷体、黑体
12. 尾巴/长条带的颜色和透明度（需开反锯齿）可以在 gamedata*.txt 中 修改 tail_color_* 实现
13. 开着CapsLock (大写指示灯)，掉血不死，但是只要血减到过0，血槽就会变黑，结束之后一律算Mistake



自定义修改：
0. 选项中的游戏界面N（N为非负整数），对应游戏目录下的gamedataN.txt，该文件指明界面皮肤

1. 要关闭波纹效果可以把游戏目录中 gamedata*.txt 里的：
Image effect_note_in pic\Press.png 256 512 256 256 0 0 0 0 255 255 255 255
改成：
Image effect_note_in pic\Press.png 600 600 16 16 0 0 0 0 255 255 255 255
让其指向png中的任何空白区域即可

2. 粒子效果为高时，如果视频太亮会自动变暗一些，这是为了让粒子效果更为突出，要禁止自动变暗，
你可以在 gamedata*.txt 中将以下行：
Float video_auto_dim 0.00085
改成
Float video_auto_dim 0.0
即可。video_auto_dim = 视频_自动_变暗

3. 要想去掉 ChanceTime 时的旋转五角星，只需把 gamedata*.txt 里的：
Int spin_angle_inc[2] 20 40
改成
Int spin_angle_inc[2] 400 400
即可。20 40 为在Cool 和 Fine 时每20/40度画一个五角星，直到超过360度为止

4. 要想去掉 Combo (连击)槽，只需将对应gamedata*.txt 中的这一行：
Image combo_gauge[13] pic\gauge.png 	0 136 55 56	6 222 54 56	255 255 255 200	56 50
改成
Image combo_gauge[13] pic\gauge.png 	0 136 55 56	6 222 54 56	255 255 255 0	56 50
