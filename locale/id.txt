\nKeyboard Keterlambatan Kompensasi\nUnit: dalam milidetik\nTIDAK mempengaruhi delay suara kunci\nTekan F5 / F6 selama pertandingan tidak sama
1. Selama pertandingan, menyalakan CapsLock untuk mencegah &#39;kematian&#39;. Namun, setelah HP sudah di bawah 0, HP pengukur akan berubah menjadi hitam, hasil akhir akan selalu &#39;KESALAHAN&#39;\n2. Selama permainan, tekan F1 ~ F4 untuk menyesuaikan dan menyimpan catatan keterlambatan peta saat ini (lihat calibration.inf dalam folder itu lagu), tekan F5 / F6 untuk menyesuaikan dan menyimpan Keyboard keterlambatan kompensasi, tekan PageUp / PageDown untuk menyesuaikan dan menyimpan volume lagu saat\n3. Tekan / - untuk menyesuaikan tempo musik (mempertahankan lapangan yang sama), tekan F11 / F12 untuk menyesuaikan lapangan musik (menjaga tempo yang sama), tekan F7 / F8 untuk menyesuaikan kecepatan gerakan ekor\n4. Ketika PitchShiftDSP dinonaktifkan, lapangan tidak dapat diubah secara independen dari tempo, (hanya sampling rate berubah), tapi latency musik / delay minimal\n5. Untuk mengubah alamat folder lagu, tambahkan baris berikut ke config.ini, songPath = [my_song_folder_path] \\n6. Folder lagu mungkin berisi sub-folder sekarang sehingga Anda dapat mengatur ulang koleksi lagu\n7. Klik pada progress bar untuk mencari jalur untuk posisi itu, tekan &lt;Tab&gt; untuk mencari jalur untuk posisi sebelumnya mencari-track; Namun, hal tersebut akan mengakibatkan &#39;KESALAHAN&#39; untuk skor akhir
